Custom desktop/executable icon:

1) Put your icon file here as ONE of:
   app-icon.png
   app-icon.jpg
   app-icon.jpeg
   app-icon.ico

2) Build the executable:
   build_desktop_exe.cmd

Installer icon (optional):

1) Put your installer icon file here as:
   installer-icon.ico

2) Build installer executable:
   build_installer_exe.cmd

Notes:
- The build script auto-generates a multi-size ICO for Windows EXEs.
- For best quality source images, use a square image at least 512x512.
